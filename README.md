# omnidirectional_mobile_platform
全向移动平台

通过Stm32f103主控读取光电对管状态，判断是否触发黑线，进而调整行进路线。再通过步进电机精确控制行走距离和方向，进而到达目标位置。
到达位置后打开涵道风扇，完成灭火任务。
